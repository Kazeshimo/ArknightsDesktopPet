# ArknightsDesktopPet
一个简单的桌面宠物

已经实现的功能：

1.双击互动且有语音

2.单击鼠标左键随意拖动

3.鼠标穿透效果

4.右键菜单快速打开浏览器和微信，以及交谈

5.睡觉，坐下，站立三个动作的切换

6.调整小人的不透明度

7.令人肝痛的bgm

8.右键退出按钮
